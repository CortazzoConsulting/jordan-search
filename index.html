Negotiation talking points

  • Anchor to the posted range — when public, ask for the upper third and cite it.
  • Lead with the 2.5-year family-business tenure as the stability proof.
  • For OTE roles, negotiate base first, then a 90-day ramp guarantee.
  • If base won't move, trade for a 6-month review, signing bonus, or PTO.
  • Never give the first number — return the range and ask their budget.
